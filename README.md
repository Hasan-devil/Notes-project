# 📝 Notes & To-Do App

A clean and responsive web application designed to help users organize their thoughts and daily tasks in one place. Built with a focus on **User Experience** and **Responsive Design**.

---

### ✨ Features

* **Dual Functionality:** Seamlessly switch between long-form notes and actionable to-do lists.
* **Responsive UI:** Optimized for both mobile and desktop screens.
* **CRUD Operations:** Create, Read, Update, and Delete notes with ease.
* **Persistent Storage:** (Optional: Mention if you used LocalStorage, Firebase, or SQLite here).
* **Search & Filter:** Quickly find specific notes or filter tasks by completion status.
